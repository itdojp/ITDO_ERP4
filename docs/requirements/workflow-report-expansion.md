# 自動化ワークフロー/レポート拡張（案）

## 目的
- 定期レポート生成・配信の運用を標準化する。
- 承認フローの条件分岐を拡張し、運用負荷を下げる。

## レポート拡張
### 定期レポート
- 月次/週次の自動生成
- 送信先: 管理者/グループ/担当者
- フォーマット: CSV/PDF

### テンプレ拡張
- レポート項目のON/OFF
- 出力レイアウトのプリセット
- 署名/ロゴの挿入（PDF）

## 自動化ワークフロー
- 条件分岐:
  - 金額閾値
  - 定期案件/小額案件
  - プロジェクト種別/顧客/部門
- 分岐アクション:
  - 承認スキップ
  - 二重チェック
  - 追加承認の要求

## 通知/監査
- 自動化によりスキップした場合も監査ログに記録
- アラートはダッシュボード/メールへ通知

## オープン事項
- 自動化条件の管理画面UI
- レポート配信の失敗時再送
