# ドキュメントテンプレートのバリエーション（たたき台）

## 背景
旧システムでは税抜/税込・日付有無・印影有無など複数テンプレートを選択運用していました。ERP4ではテンプレートは最小構成です。

## 現状（ERP4）
- `doc_template_settings` でテンプレ設定は可能
- `pdf-templates` は最小構成（estimate/invoice/purchase_order）

## 差分/課題
- 旧システムで使っていたテンプレート種別の再現範囲が未確定
- どのテンプレートを誰が/いつ選択するかの運用が未決定

## 方針案（未確定）
- テンプレートは「種類追加」ではなく「設定フラグ」で対応する案
- 税/日付/印影をテンプレの layoutConfig で制御し、切替は templateSetting で管理
