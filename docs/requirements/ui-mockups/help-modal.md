# ヘルプ/相談モーダル（ウェルビーイング）

## 目的
Not Good が続く / 相談したい人が、3クリック以内で相談先を選び、通知/案内を受けられる導線を提供する。

## 内容
- ボタン文言: 「ヘルプ / 相談したい」
- 表示場所: 日報+ウェルビーイングモジュール右上、常時表示（Not Goodが続く場合は強調）
- モーダル項目:
  1. 上長に相談（通知先: 上長グループ）
  2. 人事・労務に相談（通知先: 人事グループ）
  3. 産業医・保健スタッフ（任意: 在籍時のみ表示）
  4. 社外相談窓口（EAP等）
  5. 緊急の支援が必要かもしれない（公的窓口一覧リンク）
- 各選択肢に表示する情報:
  - 通知先（誰が見るか）
  - 目安対応時間（例: 2営業日以内）
  - 個人情報の取扱い（評価に用いない、職場環境改善・健康配慮のためのみ）
- 緊急時の案内: 「強い自殺念慮などの緊急状態では、ただちに医療機関や公的相談窓口へ」+ リンク一覧。
- 送信後: 確認メッセージ + どこに通知したかを明示。

## 権限
- モーダルを開く/送信: 全ユーザー
- 通知を閲覧/対応: 人事または指定の相談先グループ

## 留意
- 任意選択であり、匿名ではないことを明示（匿名ニーズは別途検討）。
- 表示文言に「評価には使わない」旨を含め、安心感を持たせる。
