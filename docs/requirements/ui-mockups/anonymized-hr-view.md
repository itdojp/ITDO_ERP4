# 人事向け匿名集計ビュー（骨組み）

## 目的
ウェルビーイングの結果を個人特定しない形で人事に提示し、環境改善の判断材料とする。

## 表示条件
- 集計単位: 部署/チーム/プロジェクトなど
- 最低人数: 5人未満のグループは表示しない（"データ不足"表示）

## 指標（初期）
- 直近4週間の Not Good 割合
- ストレス平均（0-4段階）
- 仕事の充実感平均（0-4段階）
- ヘルプリクエスト件数

## 画面構成（例）
- フィルタバー: 部署/チーム選択、期間選択（4週/8週）、最小人数閾値
- カード一覧: グループごとの指標カード（Not Good %, ストレスAvg, 充実感Avg)
- トレンド: 週次の Not Good % 推移（ラインチャート）
- ヒートマップ（オプション）: 部署 × 週 の色分け

## 文言
- 「個人を特定しない匿名集計です。5人未満のグループは表示しません。」
- 「評価目的での利用は禁止。マネジメント支援/環境改善のみ。」

## 後続拡張
- フィルタ: 雇用区分/勤務地/職種
- アラート: Not Good % が一定値超過で通知
- CSV/PDF エクスポート（匿名性を維持する形式）
